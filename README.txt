==================================================
 LiveScoreViewer  (Max for Live MIDI Device)
==================================================

Ableton Live のMIDIクリップを、選択／再生に合わせてリアルタイムに
五線譜で表示する Max for Live デバイスです。

主な機能:
  - クリップの音符を五線譜表示（和音・休符・付点・連桁・連符）
  - 調号の表示と臨時記号（♯ ♭ ♮）の自動処理
  - 拍子記号の表示（変拍子クリップにも対応：Scanボタンで取込）
  - クレフ選択（ト音 / ヘ音 / 大譜表 / Auto 自動判定）
  - 小節ナビ（◀ ▶）と、再生に追従するFollowモード

--------------------------------------------------
 インストール / 使い方
--------------------------------------------------
1. このZIPを解凍し、中の5つのファイルを「同じフォルダ」に置いたままにしてください。
     - LiveScoreViewer.amxd   （デバイス本体）
     - m4l_score_inline.html  （楽譜描画エンジン）
     - dict_to_json.js
     - meter_scan.js
     - follow.js
   ※ これらは必ず同じフォルダに一緒に置いてください。1つでも欠けると動きません。

2. Ableton Live で、MIDIトラックに LiveScoreViewer.amxd を読み込みます
   （フォルダから .amxd をトラックへドラッグ＆ドロップ）。

3. MIDIクリップを選択すると、五線譜が表示されます。
   - Key   : 調号を選択
   - Clef  : クレフを選択（Auto推奨）
   - Bar   : ◀ ▶ で表示小節を移動
   - Scan  : クリップ途中で拍子が変わる場合に押すと拍子を取り込みます
   - Follow: 点灯中は再生に追従して表示が進みます

--------------------------------------------------
 動作環境
--------------------------------------------------
  - Ableton Live 11 以降 + Max for Live
  - Windows / macOS
